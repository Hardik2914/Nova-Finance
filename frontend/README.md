# Nova Finance Frontend

This is the React, TypeScript, and Vite-based web client for Nova Finance.

## Development

### Prerequisites
- Node.js (v18 or v20 recommended)

### Run Server Locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run Vite dev server:
   ```bash
   npm run dev
   ```

### Compile Production Build
```bash
npm run build
```
The compiled static bundles will be output to the `dist` directory.
